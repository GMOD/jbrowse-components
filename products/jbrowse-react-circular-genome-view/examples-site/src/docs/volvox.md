`@jbrowse/react-circular-genome-view2` embeds an interactive **circular** genome
view in any React app. It shines for structural variants, chromosome-scale
rearrangements, and whole-genome summaries, where chromosome-spanning breakends
become visible chord-style across the circle. If you need a linear,
track-by-track view instead, see
[`@jbrowse/react-linear-genome-view2`](https://jbrowse.org/jb2/docs/embedded_components/).

## A minimal embed

Pass an `assembly`, a list of `tracks`, and an `init` for the tracks to open on
first paint straight to `<CircularGenomeView>` as props — no `createViewState`
call needed. The CGV config reuses the JBrowse format, with one difference: it
takes a single `assembly` rather than an `assemblies` array.

```jsx
import { CircularGenomeView } from '@jbrowse/react-circular-genome-view2'
;<CircularGenomeView
  assembly={assembly}
  tracks={tracks}
  init={{ tracks: ['volvox_sv_test'] }}
/>
```

The props are **initial values** — read once on mount. To open a track in
response to a runtime event rather than at first paint, or to reach the engine
imperatively, use the unmanaged `createViewState` flow instead — see
[show a track](../show-track/).

## Configuration reference

The autogenerated schema references list every available field: track/assembly
slots under [docs/config](https://jbrowse.org/jb2/docs/config/) and view
snapshot properties under [docs/models](https://jbrowse.org/jb2/docs/models/)
(e.g. [CircularView](https://jbrowse.org/jb2/docs/models/circularview/)).
