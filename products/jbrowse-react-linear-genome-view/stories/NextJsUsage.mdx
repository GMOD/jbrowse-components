# Usage with Next.js

[Next.js](https://nextjs.org/) is a popular framework for building
server-rendered React websites with Node.js.

In older versions, some special steps needed to be taken to run under Next.js
but it is now easy to run JBrowse 2 under next.js without any special
considerations. We keep track of example embeddings
[here](https://jbrowse.org/jb2/docs/embedded_components)
