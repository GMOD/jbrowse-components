# Horizontal flipping

Regions can be rendered reversed (3'→5'), which is essential for
viewing genes on the negative strand or for laying out synteny-style
comparisons. The view supports flipping a single locstring, an entire
view, or individual regions within a multi-region view.

## Flip via locstring

Append `[rev]` to a locstring passed to `navToLocString`. The flip
happens as part of navigation.

<Story of={HorizontallyFlippedViaLocstring} />

## Flip via button

Toggle the orientation imperatively — useful for custom toolbar
buttons or keyboard shortcuts.

<Story of={HorizontallyFlippedViaButton} />

## Flip via displayed regions

Set `reversed: true` on a `displayedRegions` entry to flip just that
region. Other regions in the same view stay in their natural
orientation.

<Story of={HorizontallyFlippedViaDisplayedRegions} />

## Mixing orientations

Combine forward and reversed regions in a single view — the building
block for gene-centric layouts (always-5'-to-3') and synteny ribbons.
