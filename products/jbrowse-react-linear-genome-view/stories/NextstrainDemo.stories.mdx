## Nextstrain SARS-CoV-2 example

We created this demo to help show how a instance can be used to browse the
SARS-CoV-2 genome similar to Nextstrain (https://nextstrain.org). It uses a
color config to color gene's similar to the Nextstrain website, and also
displays "Entropy score" to show diversity

### Source

This config makes use of the "FromConfigAdapter" where we supply the feature
data to the system through JSON rather than from flatfiles like GFF3 and
BigWig. The FromConfigAdapter can be useful to display on-the-fly analyses from
your code or to avoid having to write data to the disk e.g. generate a
BigWig/GFF3. See
products/jbrowse-react-linear-genome-view/public/nextstrain_covid.json for
details, summarized below here too

```
  const state = createViewState({
    assembly,
    tracks,
    defaultSession,
    location: 'SARS-CoV-2:1..29,903',
    onChange: patch => {
      // eslint-disable-next-line no-console
      console.log('patch', patch)
    },
    configuration: {
      theme: {
        palette: {
          primary: {
            main: '#5da8a3',
          },
          secondary: {
            main: '#333',
          },
        },
      },
    },
  })
```
