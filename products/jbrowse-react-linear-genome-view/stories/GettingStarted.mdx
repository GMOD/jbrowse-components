# Welcome

This is the [Storybook](https://storybook.js.org/) for the
[JBrowse 2 React Linear Genome View](https://www.npmjs.com/package/@jbrowse/react-linear-genome-view2),
which embeds an interactive genome browser in any React application.

Stories are fully interactive — pan, zoom, and open tracks as you would in your
own app, and click the **code** button at the bottom right to see the source.

## Recommended setup: `init`

The recommended way to embed JBrowse is via the `init` field on
`defaultSession.view`. You give it an assembly name, a starting locstring, and
the trackIds you want open on first paint — and JBrowse fills in the rest.

```js
const state = createViewState({
  assembly,
  tracks,
  defaultSession: {
    name: 'My session',
    view: {
      type: 'LinearGenomeView',
      init: {
        assembly: 'hg38',
        loc: 'chr1:11,106,077-11,261,675',
        tracks: ['ncbi-refseq-genes'],
      },
    },
  },
})
```

The story below is exactly this pattern, wired up to GRCh38 with a real RefSeq
gene track:

<Story of={WithInit} />

`init.loc` accepts any locstring, including space-separated multi-region strings
(e.g. `'chr1:100-200 chr1:500-600'`). `init.tracks` is a plain list of trackIds
to open.
