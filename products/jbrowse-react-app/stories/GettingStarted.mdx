# Welcome!

This site contains the [Storybook](https://storybook.js.org/) docs for the
[JBrowse 2 React App Component](https://www.npmjs.com/package/@jbrowse/react-app2).
The component makes it possible to embed the full JBrowse 2 application in any
React application.

## Required CSS import

The component uses [dockview](https://dockview.dev/) for its layout. You must
import its stylesheet in your app's entry point:

```js
import 'dockview-react/dist/styles/dockview.css'
```

## Quick start with `init`

The `defaultSession.views[*].init` shorthand is the recommended way to open a
specific location with tracks on startup:

```js
import { createViewState, JBrowseApp } from '@jbrowse/react-app2'

const state = createViewState({
  config: {
    assemblies: [
      /* ... */
    ],
    tracks: [
      /* ... */
    ],
    defaultSession: {
      views: [
        {
          type: 'LinearGenomeView',
          init: {
            assembly: 'hg38',
            loc: 'chr1:1,000,000-1,100,000',
            tracks: ['my_genes_track'],
          },
        },
      ],
    },
  },
})

return <JBrowseApp viewState={state} />
```

This works for all view types — see
[Synteny Views](?path=/docs/synteny-views--docs) for a `LinearSyntenyView`
example. The full list of `createViewState` options is in the
[API Reference](?path=/docs/api-reference--docs).
