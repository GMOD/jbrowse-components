# Theming

Configure via `configuration.theme` in the config object, following
[MUI theme palette](https://mui.com/material-ui/customization/theming/) conventions:

```js
createViewState({
  config: {
    configuration: {
      theme: {
        palette: {
          mode: 'dark',            // dark mode
          primary: { main: '#4d9221' },
          secondary: { main: '#c51b7d' },
        },
      },
    },
    // ...
  },
})
```
