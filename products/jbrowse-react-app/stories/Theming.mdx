# Theming

Configure via `configuration.theme` in the config object, following
[MUI theme palette](https://mui.com/material-ui/customization/theming/)
conventions. See the
[theming guide](https://jbrowse.org/jb2/docs/config_guides/theme/) for the full
set of options including DNA base colors:

> > > > > > > origin/main

```js
createViewState({
  config: {
    configuration: {
      theme: {
        palette: {
          mode: 'dark',
          primary: { main: '#4d9221' },
          secondary: { main: '#c51b7d' },
        },
      },
    },
    // ...
  },
})
```
