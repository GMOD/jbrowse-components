# API Reference

> For the lightweight linear-only component, see the
> [@jbrowse/react-linear-genome-view2 API Reference](https://jbrowse.org/storybook/lgv/main/?path=/docs/api-reference--docs).

## `createViewState(options)`

```js
createViewState({
  config,             // required — full app config (same format as jbrowse-web config.json)
  plugins,            // PluginConstructor[] — inline or imported plugin classes
  makeWorkerInstance, // () => Worker — web worker factory for WebWorker RPC
  onChange,           // (patch, reversePatch) => void — called on every MST patch
})
```

### Config shape

```js
{
  configuration: { /* global settings, e.g. theme */ },
  assemblies: [ /* assembly configs */ ],
  tracks: [ /* track configs */ ],
  defaultSession: { /* optional: views and widgets to open on load */ },
}
```

### `defaultSession.views[*].init` shorthand

```js
defaultSession: {
  name: 'My session',
  views: [{
    id: 'view1',
    type: 'LinearGenomeView',
    init: {
      assembly: 'hg38',
      loc: 'chr1:1,000,000-1,100,000',
      tracks: ['trackId1', 'trackId2'],
      tracklist: true,  // open track selector drawer on load
    },
  }],
}
```

Works for all view types — `LinearSyntenyView`, `DotplotView`, etc. each have their own `init` shape.

## State object

```js
// Register a track config at runtime (call from an event handler, not during render)
state.jbrowse.addTrackConf(trackConf)

// Views
state.session.views                              // array of open view models
state.session.views[0].showTrack(trackId)
state.session.views[0].hideTrack(trackId)
await state.session.views[0].navToLocString(loc, assemblyName)

// Plugin manager
import { getEnv } from '@jbrowse/core/util'
const { pluginManager } = getEnv(state)
```

### Launching views via extension points

```js
await pluginManager.evaluateAsyncExtensionPoint('LaunchView-LinearGenomeView', {
  session: state.session,
  assembly: 'hg38',
  loc: 'chr1:1-100000',
  tracks: ['trackId1'],
})
```

Other extension points: `LaunchView-LinearSyntenyView`, `LaunchView-DotplotView`.

## `loadPlugins(definitions)`

```js
import { loadPlugins, createViewState } from '@jbrowse/react-app2'

const plugins = await loadPlugins([
  { name: 'MyPlugin', url: 'https://example.com/my-plugin.js' },
])
const state = createViewState({ config, plugins })
```
