# Synteny Views

Use `type: 'LinearSyntenyView'` or `type: 'DotplotView'` in `defaultSession.views`:

```js
defaultSession: {
  views: [{
    id: 'synteny_view',
    type: 'LinearSyntenyView',
    init: {
      views: [{ assembly: 'hg19' }, { assembly: 'hg38' }],
      tracks: [['hg19_to_hg38_chain']],
    },
  }],
}
```
