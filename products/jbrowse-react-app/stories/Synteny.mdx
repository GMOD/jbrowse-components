# Synteny Views

Use `type: 'LinearSyntenyView'` or `type: 'DotplotView'` in
`defaultSession.views`:

```js
defaultSession: {
  views: [{
    id: 'synteny_view',
    type: 'LinearSyntenyView',
    init: {
      views: [{ assembly: 'volvox' }, { assembly: 'volvox_del' }],
      tracks: ['volvox_del.paf'],
    },
  }],
}
```
