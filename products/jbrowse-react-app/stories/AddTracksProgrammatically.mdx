# Adding Tracks Programmatically

Call `addTrackConf` and `showTrack` from an event handler — not during render:

```js
state.jbrowse.addTrackConf({
  type: 'FeatureTrack',
  trackId: 'my_genes',
  name: 'My Genes',
  assemblyNames: ['hg38'],
  adapter: { type: 'Gff3TabixAdapter', uri: 'https://example.com/genes.gff.gz' },
})
state.session.views[0]?.showTrack('my_genes')
```
