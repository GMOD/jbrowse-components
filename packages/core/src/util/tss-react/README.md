# tss-react (vendored)

Vendored subset of [tss-react](https://github.com/garronej/tss-react) by
Garronej, providing `makeStyles`/`cx`/`css` on top of Emotion.

Adapted for this codebase — see inline `NOTE:` comments for deliberate
deviations from upstream (e.g. `cssAndCx.ts` omits the issue-#27 `cx` workaround
this codebase doesn't need).
