Visualizing alignments is an important aspect of genome browsers. This guide will go over the main features of the "Alignments track"

### Alignments track

JBrowse alignments track feature a pileup and a coverage visualization

The pileup is a standard view of the alignments mapped to the genome, each box representing a "read" from the sequencer
