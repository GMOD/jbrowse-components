## How to Use This Archive

- **Looking for why a decision was made?** Check the corresponding ARCHIVE file for context and alternatives considered.
- **Researching a feature?** ARCHIVE files may contain earlier exploration that was deprioritized.
- **Cleaning up old notes?** If a file is clearly resolved (e.g., debug logs for fixed bugs), it can be deleted.

Do **not** commit changes to ARCHIVE files — treat them as read-only documentation of the project's evolution.
