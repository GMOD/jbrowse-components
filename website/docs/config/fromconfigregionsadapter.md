title: FromConfigRegionsAdapter
toplevel: true
---
used for specifying refNames+sizes of an assembly
#### slot: features
```js

    /**
     * !slot
     */
    features: {
      type: 'frozen',
      defaultValue: [],
    }
```
#### slot: featureClass
```js

    /**
     * !slot
     */
    featureClass: {
      type: 'string',
      defaultValue: 'SimpleFeature',
    }
```
