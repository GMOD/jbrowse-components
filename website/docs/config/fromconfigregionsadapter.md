used for specifying refNames+sizes of an assembly

### FromConfigRegionsAdapter - Slots

#### slot: features

```js
features: {
      type: 'frozen',
      defaultValue: [],
    }
```

#### slot: featureClass

```js
featureClass: {
      type: 'string',
      defaultValue: 'SimpleFeature',
    }
```
