## Init Field

Configure the view at startup using the `init` field:

```javascript
const state = createViewState({
  assembly,
  tracks,
  defaultSession: {
    name: 'My Session',
    view: {
      type: 'LinearGenomeView',
      init: {
        loc: 'chr1:11,106,077-11,261,675',
        assembly: 'hg38',
        tracks: ['track-id-1', 'track-id-2'],
        tracklist: true,  // Show track selector
        nav: true,        // Show navigation bar
      },
    },
  },
})
```

### Init Options
- `loc?: string` — Initial location
- `assembly: string` — Assembly name (required)
- `tracks?: string[]` — Track IDs to display
- `tracklist?: boolean` — Show hierarchical track selector
- `nav?: boolean` — Show navigation header
- `highlight?: string[]` — Genomic regions to highlight

**Example:** `WithInit`

## Default Session

Specify the initial view state:

```javascript
defaultSession: {
  name: 'My Session',
  view: {
    id: 'linearGenomeView',
    type: 'LinearGenomeView',
    offsetPx: 2000,        // Initial scroll position
    bpPerPx: 0.05,         // Zoom level
    displayedRegions: [
      {
        refName: 'ctgA',
        start: 0,
        end: 50001,
        assemblyName: 'volvox',
      },
    ],
    tracks: [],  // Initially visible tracks
  },
}
```

**Example:** `DefaultSession`

## Real Genome Data

Initialize with human genome (GRCh38):

```javascript
const assembly = {
  name: 'hg38',
  sequence: {
    type: 'ReferenceSequenceTrack',
    adapter: {
      type: 'BgzipFastaAdapter',
      fastaLocation: { uri: 'https://...' },
      faiLocation: { uri: 'https://...' },
      gziLocation: { uri: 'https://...' },
    },
  },
}

const state = createViewState({
  assembly,
  tracks: geneTrackConfigs,
  defaultSession: { /* ... */ },
})
```

**Examples:** `WithInit`, `HumanExomeExample`
