## Navigating Regions

### Location Strings

Navigate using simple location string syntax:

```javascript
const state = createViewState({
  assembly,
  tracks,
  location: 'chr1:1000..2000',  // RefName:start..end
})
```

**Example:** `ExternalNavigateLocstring`

### Location Objects

Use location objects for programmatic navigation:

```javascript
const location = {
  refName: 'chr1',
  start: 1000,
  end: 2000,
  assemblyName: 'hg38',
}

await state.session.view.navToLocString(location)
```

**Example:** `ExternalNavigateObject`, `UsingLocObject`

## Reversed Orientation

Display region in reverse complement using `[rev]` suffix:

```javascript
const state = createViewState({
  assembly,
  tracks,
  location: 'chr1:1000..2000[rev]',
})
```

Or programmatically:

```javascript
state.session.view.displayedRegions = [
  {
    refName: 'chr1',
    start: 1000,
    end: 2000,
    reversed: true,  // Reverse complement
    assemblyName: 'hg38',
  },
]
```

**Examples:** `HorizontallyFlippedViaLocstring`, `HorizontallyFlippedViaButton`, `HorizontallyFlippedViaDisplayedRegions`

## Multiple Regions

Display multiple genomic regions in one view:

```javascript
state.session.view.displayedRegions = [
  { refName: 'chr1', start: 0, end: 10000, assemblyName: 'hg38' },
  { refName: 'chr2', start: 0, end: 10000, assemblyName: 'hg38' },
]
```

**Example:** `WithMultipleDisplayedRegionsFlipped`

## Listen to Navigation Changes

```javascript
import { autorun } from 'mobx'

// React to region changes
autorun(() => {
  const regions = state.session.view.displayedRegions
  console.log('User navigated to:', regions)
})
```

**Examples:** `WithObserveVisibleRegions`, `WithObserveVisibleFeatures`
