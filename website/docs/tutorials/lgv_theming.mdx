## Custom Colors

Customize UI and DNA base colors:

```javascript
const state = createViewState({
  assembly,
  tracks,
  configuration: {
    theme: {
      palette: {
        primary: { main: '#1976d2' },
        secondary: { main: '#dc004e' },
        bases: {
          A: { main: '#98FB98' },  // Green
          C: { main: '#87CEEB' },  // Blue
          G: { main: '#DAA520' },  // Gold
          T: { main: '#DC143C' },  // Red
        },
      },
    },
  },
})
```

**Example:** `WithCustomTheme`

## Dark Mode

Enable dark theme:

```javascript
const state = createViewState({
  assembly,
  tracks,
  configuration: {
    theme: {
      palette: {
        mode: 'dark',
      },
    },
  },
})
```

**Example:** `WithDarkTheme`

## External Styling

Apply CSS to the container:

```javascript
export function App() {
  const state = createViewState({ assembly, tracks })
  
  return (
    <div style={{
      height: '100vh',
      fontFamily: 'Arial, sans-serif',
      backgroundColor: '#f5f5f5',
    }}>
      <JBrowseLinearGenomeView viewState={state} />
    </div>
  )
}
```

**Example:** `WithOutsideStyling`

## Dynamic Theme Toggle

Change theme at runtime:

```javascript
const [isDark, setIsDark] = useState(false)

const theme = {
  theme: { palette: { mode: isDark ? 'dark' : 'light' } }
}

const state = createViewState({
  assembly,
  tracks,
  configuration: theme,
})
```
