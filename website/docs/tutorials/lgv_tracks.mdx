## Display Specific Tracks

Show tracks at startup:

```javascript
const state = createViewState({
  assembly,
  tracks,
  defaultSession: {
    name: 'Session',
    view: {
      type: 'LinearGenomeView',
      init: {
        assembly: 'volvox',
        tracks: ['volvox-alignments', 'volvox-genes'],
      },
    },
  },
})
```

Or programmatically:

```javascript
state.session.view.showTrack('volvox-alignments')
```

**Example:** `WithShowTrack`

## Disable Track Addition

Prevent users from adding new tracks:

```javascript
const state = createViewState({
  assembly,
  tracks,
  disableAddTracks: true,
})
```

**Example:** `DisableAddTrack`

## Track Selector (Drawer)

Show hierarchical track selector in drawer:

```javascript
defaultSession: {
  name: 'Session',
  view: {
    type: 'LinearGenomeView',
    init: {
      assembly: 'volvox',
      tracklist: true,  // Show track selector drawer
    },
  },
}
```

**Example:** `WithDrawerWidget`

## Observable Tracks

React to track visibility changes:

```javascript
import { autorun } from 'mobx'

autorun(() => {
  const visibleTracks = state.session.view.tracks
  console.log('Visible tracks:', visibleTracks)
})
```

**Example:** `WithObserveVisibleFeatures`
