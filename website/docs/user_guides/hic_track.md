Visualizing Hi-C data can be performed with .hic files, a format produced by
Juicer/Juicebox and many other Hi-C pipelines. JBrowse uses the hic-straw
module developed by the Juicebox/igv.js team to read these files.

Currently configuration options are basic for Hi-C tracks, see
[the comprehensive config guide](/docs/config_guides/hic_track/) for info about
configuring Hi-C tracks.
