This release fixes some issues where the alignments track was not scrollable and
other misc issues

## Downloads

- [v4.1.15](https://github.com/GMOD/jbrowse-components/releases/tag/v4.1.15)

To install JBrowse 2 for the web, you can download the link above, or you can
use the JBrowse CLI to automatically download the latest version. See the
[JBrowse web quick start](https://jbrowse.org/jb2/docs/quickstart_web) for more
details.

  ([#5513](https://github.com/GMOD/jbrowse-components/pull/5513)) @cmdcolin
