Highlights

## New features

- When users of jbrowse-web open files from their local machine, it can continue
  to work after page refresh. This is currently a Chrome only feature
  facilitated by the FileSystemFileHandle API
- Add built-in linkage disequilibrium viewer for multi-sample variant tracks.
  This can load data from multi-sample variant files or PLINK output files
- Allow custom jexl expressions to be used in feature filters
- CRAM optimizations
- Better ESM compatibility

## Notable bugfixes

- Large VCF files not displaying
- Windows desktop app text indexing
