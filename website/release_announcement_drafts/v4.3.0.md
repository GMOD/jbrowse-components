This release adds a variety of small bugfixes

- Loads bedMethyl as multiquantitativetrack automatically
- Improved/unified behavior of the &highlight= matching the bookmark highlight
  behavior
- Fixed specification of multi-level synteny views from URL query params (now
  takes a two dimensional array of string[][], where each pairwise comparison
  level takes a list of syntenytrack trackIds)
- Add featureHeight param for alignmentstracks in jb2image CLI tool
- Website overhaul! Uses astro instead of docusaurus now
- Standardized discordant pair coloring with IGV based coloring
  (navy,green,teal) Let us know if you have any feedback

Additionally, the new GPU refactor is ongoing and we are interested in hearing
from users that would like to test. Please contact jbrowse2@berkeley.edu for
more info or visit https://github.com/GMOD/jbrowse-components/pull/5468 for an
overview/comment thread
