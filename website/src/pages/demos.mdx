# JBrowse 2 demos

These demos contain examples of new workflows and views in JBrowse 2, with links
to live sessions on the web so you can try them out in the app
yourself. The link above will always direct to the most recent demo that has
been released.

Note: everything demonstrated here on the web can also be done in JBrowse
desktop.
